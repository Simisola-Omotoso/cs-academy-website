This website showcases a variety of courses from different topics in computer science. These courses are meant to be taken by high school students or beginners.

## About
CS Academy is a computer science educational platform that creates free courses for students interested in the field. We seek to provide content for beginners who are overwhelmed in getting started in computer science.

![office](https://github.com/user-attachments/assets/850b9a44-99bf-4175-b482-411d033f3bf5)

## Home Page
Home page displays a few courses that the user can get started with, mainly in software engineering: front-end, back-end, full-stack, and web development. The course thumbnails can be clicked on which will take you to the actual courses themselves.

## Topics
